# vue_ts
本项目主要以一个记事本便利贴的方式熟悉在vue中如何TypeScript.主要功能包括：笔记的增删改查以及筛选功能。
功能比较简单，主要熟悉ts的应用。
数据是保存本地的，所以需要自己手动添加;
本项目作为ts+vue练手项目，适用于了解ts基础语法的人群.
## Project setup
```
npm install
```
